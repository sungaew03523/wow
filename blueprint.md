
# B.A.H. - Blueprint

## Обзор

B.A.H. (Bidding Auction House) — это мобильное приложение для отслеживания аукционов в World of Warcraft. Оно предоставляет пользователям мощные инструменты для анализа цен, отслеживания избранных предметов и определения наиболее выгодных ремесленных формул.

## Реализованные функции

### Версия 1.0.0

*   **Аутентификация и подключение к API Blizzard:** Настроен безопасный доступ к данным аукциона.
*   **Отображение данных аукциона:**
    *   Создан экран `AuctionHouseScreen` для отображения списка предметов с аукциона.
    *   Реализован поиск и фильтрация по названию.
    *   Добавлена возможность добавлять предметы в избранное.
*   **Отслеживание избранного:**
    *   Создан экран `FavoritesScreen` для просмотра сохраненных предметов.
    *   Реализовано сохранение избранного в `Firestore`.
*   **Расчет прибыльности ремесел:**
    *   Создан экран `FarmsScreen` для управления ремесленными формулами.
    *   Реализовано добавление, удаление и редактирование "фармов" (ремесленных формул).
    *   Формулы хранятся в `Firestore`.
