
# Blueprint: Wow Gold Tracker

## Обзор

Это Flutter-приложение для отслеживания цен на аукционе World of Warcraft. Оно позволяет пользователям искать предметы, добавлять их в избранное и отслеживать динамику цен на них с помощью интеграции с Blizzard API и Firebase.

## Реализованные функции и дизайн

### **Версия 1.0.0**

*   **Главный экран (`HomeScreen`):**
    *   Показывает текущую цену WoW Token.
    *   Отображает список предметов, доступных для добавления в избранное.
    *   Реализован поиск по названию предмета.
    *   Пользователи могут добавлять и удалять предметы из избранного.

*   **Экран избранного (`FavoritesScreen`):**
    *   Отображает список избранных предметов с их иконками и текущими ценами (минимальная, рыночная, средневзвешенная).
    *   Для каждого предмета отображается **график изменения средневзвешенной цены**.
    *   Реализована функция **обновления цен** для всех избранных предметов по нажатию кнопки.
    *   Добавлена возможность **переименования дубликатов** предметов для удобства отслеживания.
    *   Для каждого предмета можно **настроить индивидуальный объем для анализа рынка** (по умолчанию 1000). Это значение влияет на расчет средневзвешенной цены.

*   **Навигация:**
    *   Используется `go_router` для навигации между главным экраном и экраном избранного.

*   **Бэкенд и API:**
    *   **Firebase Firestore:** Используется для хранения списка избранных предметов, их цен и истории цен.
    *   **Blizzard API:** Интегрирован для получения:
        *   Цены WoW Token.
        *   Списка предметов с аукциона.
        *   Актуальных цен на реагенты.

*   **Стиль и дизайн:**
    *   Приложение использует кастомную тему, вдохновленную стилистикой World of Warcraft, с преобладанием золотых и темных тонов.
    *   Используется кастомный шрифт `Montserrat`.
    *   Кнопки и интерактивные элементы имеют стилизованную золотистую рамку.
    *   Карточки предметов имеют тень для создания эффекта глубины.
    *   Иконки предметов загружаются напрямую из Blizzard API, что обеспечивает аутентичный вид.
