
# Blueprint: WoW Auction House UI

## Обзор

Это приложение представляет собой интерфейс для просмотра предметов из аукционного дома World of Warcraft. Оно позволяет пользователям загружать список предметов из Blizzard API, просматривать их, добавлять в избранное и отслеживать актуальные цены.

## Структура и навигация

Приложение использует пакет `go_router` для навигации и состоит из двух основных экранов:

1.  **`/` (Главный экран):** `AuctionHouseScreen`
    *   Отображает полный список доступных предметов, полученных из Blizzard API.
    *   Позволяет пользователям искать предметы по названию.
    *   Пользователи могут добавлять и удалять предметы из избранного.

2.  **`/favorites` (Экран избранного):** `FavoritesScreen`
    *   Отображает только те предметы, которые были добавлены в избранное.
    *   Позволяет пользователям обновить цены на избранные предметы.
    *   Предоставляет возможность настроить параметр `countForAverage` — количество лотов, по которым рассчитывается средняя цена.

## Функциональность

- **Загрузка данных:**
  - Приложение получает данные о предметах и ценах с помощью `BlizzardApiService`, который взаимодействует с Blizzard API.
  - Загруженные предметы хранятся в коллекции `items` в Firestore.

- **Избранное:**
  - Избранные предметы хранятся в коллекции `favorites` в Firestore.
  - Цены на избранные предметы могут быть обновлены вручную.

- **Пользовательский интерфейс:**
  - Дизайн выполнен в темной теме, стилизованной под интерфейс World of Warcraft.
  - Используется пакет `google_fonts` для создания соответствующей атмосферы.

## План доработки

- **Реализовать категории:** Добавить функциональность для боковой панели категорий.
- **Сортировка и фильтрация:** Дать пользователям возможность сортировать и фильтровать списки предметов.
- **Улучшение UX:** Добавить более информативные сообщения о загрузке и ошибках.
